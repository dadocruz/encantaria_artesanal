# Fotos dos produtos

Coloque aqui as fotos reais dos produtos (ex.: `a-bruxa-do-amor.jpg`).

Depois, preencha o campo `images` do produto em `src/data/products.ts`
(ou, no futuro, pela tabela `products` do Supabase) com o caminho público,
por exemplo: `/products/a-bruxa-do-amor.jpg`.

Enquanto não houver foto, o site exibe uma ilustração premium gerada por
CSS/SVG via `ProductMockImage`, de acordo com a categoria.
